# JWT vs PASETO – Secure Token Design Explained

This repository explains **JWT (JSON Web Token)** and **PASETO (Platform-Agnostic Security Tokens)** in a clear, step-by-step manner.

JWT is widely used but easy to misuse.  
PASETO was designed to solve JWT’s security pitfalls.

## What You Will Learn
- What JWT is and how it works
- Common JWT security problems
- Why PASETO was introduced
- How PASETO fixes JWT issues
- When to use JWT vs PASETO
